


= Задание
Необходимо создать нейронную сеть, которая сможет предсказывать следующий шаг в игре "Жизнь" (https://ru.wikipedia.org/wiki/Игра_«Жизнь»)

= Используемые технологии
Код должен быть написан на Python3. Возможно использование Keras. Разрешается использовать сторонние библиотеки, доступные через pip.


= Входные данные
Программа должна представлять собой скрипт (или набор скриптов) на языке Python 3. В параметрах запуска должен указываться файл с датасетом, и параметры поля игры "Жизнь" (width, heights).

Для генерации датасета может использоваться скрипт gol_dataset.py.

Для обученной нейросети необходимо написать тест, который покажет, что сеть правильно (или нет) предсказывает следующий ход.

= Технические ограничения:
Скрипты обучения и теста обученной сети должны запускаться на Ubuntu 16.04 или MacOS. Скрипт должен позволять обучать сеть для разных значений width и heights. Скрипт должен выводить архитектуру созданной сети.
